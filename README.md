## Running the UI

1. Navigate to the ui folder:
   ```
   cd frontend
   ```

2. Install dependencies:
   ```
   npm install
   ```

3. Run the development server:
   ```
   npm run dev
   ```

The UI will be available at http://localhost:3000.

## Running the API

1. Navigate to the backend folder:
   ```
   cd backend
   ```

2. Run the Go application:
   ```
   go run .
   ```
